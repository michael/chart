Description

CHART™ is a simple charting library based on jQueryUI and the Processing.js visualization framework. It's goal is not to provide a highly customizable widget (in this case you should rather take a look at Highcharts), instead it just uses the raw data to derive suitable plotting options on its own. Source code is available at Github.
